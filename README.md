# Gym Management System 🏋️

A Java Swing desktop app to manage gym operations such as:

- 🧍 Member Registration
- 📋 Trainer Management
- 📅 Attendance Tracking
- 💸 Payment Records
- 📆 Weekly Schedules

## 🛠 Technologies
- Java 8
- Swing & AWT
- MySQL (via JDBC)
- IntelliJ IDEA
