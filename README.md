# 💪 Gym Management System

A Java Swing desktop application to manage gym operations:

### ✨ Features
- 🧍 Member Registration
- 💼 Trainer Management
- 📅 Attendance Tracking
- 💸 Payment Handling
- 📆 Weekly Schedules

### 🛠 Built With
- Java (Swing + AWT)
- MySQL (via JDBC)
- IntelliJ IDEA

### 🚀 How to Run

#### 1. Compile the code
```bash
javac -d out -cp "lib/*" src/*.java

```

#### 2. Run the main class
```bash
java -cp "out;lib/*" GymAdmin
```
